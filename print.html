PRINT PROJECT

The goal of this project was to design, print, and finish a custom bookmark for Pequea Elementary School Library

This project required one to brainstorm, sketch, design, print, and cut to create a bookmark. Our designs were created in Adobe InDesign. We used the Offset Lithography printing method to create our bookmarks. During this project we learned how the printing plates are created and how the Offset Lithography printing machines works.
